# 디지털 기초 수업 실습 앱

발달장애 성인 대상 "디지털 기초" 수업에서 학생들이 각자 노트북으로 직접 실습하는 웹앱입니다.

- **학생 접속 주소**: https://hyunminju.github.io/digitalbasic/
- 단일 HTML 파일(`lesson1.html`), 외부 라이브러리 없음, 인터넷만 되면 어떤 브라우저에서든 동작

## 1차시 구성

| 모듈 | 단계 |
|---|---|
| ⏻ 전원 켜고 끄기 (7단계) | 전원 버튼 찾아 누르기 → 부팅 기다리기 → 바탕화면 알기 → "끌 때는 화면에서!" 개념 → 시작 버튼 → 전원 → 시스템 종료 |
| 🖱️ 마우스 연습 (8단계) | 역할 소개 → 별에 올리고 3초 기다리기 → 클릭 → 더블클릭(선물 열기) → 오른쪽 클릭(비밀 메뉴) → 스크롤(꿀벌 옮기기) → 드래그(과일 4개) → 자유 연습장 |
| ⌨️ 키보드 연습 (5단계) | 역할 소개 → 아무 키나 눌러보기 → 스페이스 → 엔터 → 내 이름 첫 글자 자유 놀이 |

세 모듈을 모두 완료하면 "오늘 공부 다 했어요! 🎓" 축하 화면이 나옵니다.

## 설계 원칙

- 실패 없음: "틀렸어요" 표시 없이 부드러운 음성 안내로만 교정
- 한 화면 한 행동, 큰 타깃(88px+)·큰 글씨·고대비(빔프로젝터 대응)
- 모든 지시문 음성(TTS) 자동 낭독 + 소리 크게/작게/끄기 + 다시 듣기
- 시간 압박 없음, 성공마다 칭찬, 모듈 완료 배지 저장
- 한글 IME 상태와 무관하게 키보드 판정(`e.code` 기준), 터치패드 완전 지원
